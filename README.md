# M720Q-Hackintosh-Ventura
联想M720Q小主机Ventura 13.3.1

处理器：i9-9900T ES

主板：联想312D (B360芯片组)

硬盘：长城 GW3500 512G

内存：光威 DDR4 2666 16G x2

显卡：UHD 630

无线网卡：AC9560


无线正常，蓝牙正常，投送无法测试应该是无法投送。
小主机自配的TYPE-C接口视频也能4K输出正常，USB口目前测试也正常。

其他亲自己生成GenSMBIOS。即可正常使用。目前使用半月无明显BUG。
